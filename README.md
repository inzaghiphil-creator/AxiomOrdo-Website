# AxiomOrdo Website

Deterministic regulatory assurance for evidence-heavy compliance.

## Overview

This repository contains the AxiomOrdo parent brand landing page and product pages for PFAS and Golden Thread.

## Tech Stack

- **Framework**: Vite + React 19 + TypeScript
- **Styling**: Tailwind CSS v4
- **Routing**: React Router
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Fonts**: Cormorant Garamond (serif), IBM Plex Sans (sans-serif)

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

Deployed on Vercel. The site uses client-side routing with SPA configuration.

## Structure

- `/` - Parent brand landing page
- `/pfas` - PFAS product page
- `/golden-thread` - Golden Thread product page

## Design Principles

- Restrained premium aesthetic
- Black logo, warm white/stone background
- Evidence-bound messaging discipline
- Clear distinction between capability and maturity
